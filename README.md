# QR Code Generator

Проект представляет собой веб-приложение для генерации QR-кодов.  
Он позволяет пользователю:

1. Преобразовывать **текст или URL** в QR-код через внешний API.
2. Преобразовывать **изображение в QR-код**, генерируя QR-код с ссылкой на изображение через imgbb.com API (без использования серверной части).

Технологии

- HTML5, CSS3, JavaScript
- [QRCode.js](https://github.com/davidshimjs/qrcodejs) для локальной генерации QR-кода
- [QR Server API](https://goqr.me/api/) для генерации QR-кода по тексту/URL
- [ImgBB API](https://api.imgbb.com/) для загрузки изображений и получения ссылки

#  Функционал

# 1. Генерация QR-кода из текста или URL
- Ввод текста или ссылки в первое поле
- Нажатие кнопки "Сгенерировать QR код" или клавиши Enter
- Появляется QR-код, который можно использовать

# 2. Генерация QR-кода из изображения
- Выбор изображения через файловый ввод
- Нажатие кнопки "Загрузить и создать QR код"
- Изображение загружается на ImgBB, формируется QR-код с ссылкой на изображение
- QR-код отображается в отдельном окне

# Установка и использование

1. Клонировать репозиторий:

git clone https://github.com/<your-username>/<repo-name>.git

2. Открыть index.html в браузере.

Для работы второй формы (генерация QR-кода с изображением) необходимо:

Получить API-ключ на ImgBB

Вставить ключ в переменную IMGBB_API_KEY в index.html
